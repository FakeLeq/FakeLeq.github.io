Start

You need in-game assets like item sprites and items.dat
For auto getting check out Get Assets Auto

Features

Working, New, Disabled

Get Assets Auto

Description: It getting .rttex and items.dat files from your Local Growtopia directory then converting .rttex files to .png

Warn: Growtopia should be installed to auto get assets

Example Usage:

                
const assets = new GetAssets().get(usedAssets);
// usedAssets can be "flags", "items.dat", "game"

Item Scanner

Description: Get screenshot of player's view worn clothes dialog to get him/her items fast

Example Usage:

                
const scan = new Scanner("image.png", ItemData);
const get = await scan.getItems(); // returns BodyPart object

Flags

Description: You can add country or guild flags near to your name

Example Usage:

                
await renderPlayer.setFlagPath("path_to_your_flag_assets/");
await renderPlayer.renderFlag({ flagType: Types.flagTypes.GUILD, background: 658, foreground: 2, countryCode: "jp" })
// flagType can be Types.flagTypes.COUNTRY too

Custom Items

Description: We are supporting custom items, you can add your item to your character

Warn: Sadly only accepting 32x32 items

Example Usage:


citem.json 
                 
{
    "items": [
        {
            "item_id": -1, // go minus to prevent normal items
            "item_name": "Purple Bubble Wings",
            "item_texture": "../../your_image.png", // your custom item image location
            "textureX": 0, //image x
            "textureY": 0 // image y
        }
    ]
}
index.js: 
 
const ItemData = await new ItemsDat(readFileSync("./Assets/items.dat"), readFileSync("citem.json")).parse();
                                                                               // citem.json location
